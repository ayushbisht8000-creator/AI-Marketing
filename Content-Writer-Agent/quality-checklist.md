# Quality Checklist

Before finalizing content verify:

## Readability

✓ 7th–8th grade reading level

✓ Short paragraphs

✓ Human tone

✓ Active voice

## Engagement

✓ Strong introduction

✓ Logical flow

✓ Reader-focused explanations

✓ Strong conclusion

## Human Writing

✓ No robotic language

✓ No repetitive phrases

✓ Natural transitions

✓ Conversational style

## Content Quality

✓ Helpful

✓ Actionable

✓ Trustworthy

✓ Well-structured

## Final Review

✓ Sounds like an experienced human writer

✓ Easy to read

✓ Engaging from start to finish
