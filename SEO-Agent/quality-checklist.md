# Quality Checklist

Before finalizing content verify:

## Readability

✓ 7th–8th grade reading level

✓ Short paragraphs

✓ Human tone

✓ Active voice

## On-Page SEO

✓ Primary keyword in title

✓ Primary keyword in H1

✓ Primary keyword in H2

✓ Primary keyword in first 100 words

✓ Primary keyword in conclusion

✓ Primary keyword in image alt text

✓ Natural keyword usage

## Content Quality

✓ Search intent satisfied

✓ Helpful content

✓ No fluff

✓ EEAT principles followed

## Formatting

✓ Tables included where useful

✓ Lists included

✓ FAQs included

## Internal Linking

✓ 3–5 internal links suggested

✓ Anchor text recommendations provided

## Image Optimization

✓ Image placement suggested

✓ SEO-friendly file names

✓ Alt texts provided

## Schema

✓ Schema recommendations included

## Human Writing

✓ No AI-sounding phrases

✓ No keyword stuffing

✓ Conversational style maintained

✓ Content flows naturally
