# Image SEO Rules

## Objective

Optimize images to improve user experience, accessibility, page speed, and search engine visibility.

## Image Placement

Recommend images naturally throughout the content.

Suggested placements:

- After the introduction
- Between H2 sections
- Before long sections of text
- Near tables or lists
- Before the conclusion when appropriate

Avoid placing too many images close together.

## Image Quantity

For articles between 1000 and 1200 words:

- Recommend 3 to 5 images.
- Increase image count naturally for longer content.

## Image File Names

Use:

- Short, descriptive, hyphen-separated file names.
- Lowercase letters.

Examples:

- primary-keyword-guide.jpg
- benefits-of-primary-keyword.webp
- primary-keyword-checklist.png

Avoid:

- IMG001.jpg
- screenshot123.png
- image-final-final.jpg

## Alt Text Rules

Write descriptive alt text.

Requirements:

- Clearly describe the image.
- Keep alt text natural.
- Include the primary keyword naturally in at least one image alt text.
- Avoid keyword stuffing.
- Keep alt text concise and meaningful.

Examples:

Good:

- Example showing primary keyword optimization process
- Infographic explaining primary keyword strategy

Bad:

- Primary keyword primary keyword primary keyword

## Image Format Recommendations

Prefer:

- WebP for best performance
- JPG for photographs
- PNG for graphics and screenshots
- SVG for logos and icons

## Image Dimensions

Recommend appropriate image sizes.

Examples:

- Featured image: 1200×628
- Blog images: 1200×800
- Infographics: 1200×1800

## Compression

Optimize images before uploading.

Recommend:

- Compressed files
- Fast-loading images
- High-quality but lightweight formats

## Accessibility

Images should:

- Include descriptive alt text.
- Support screen readers.
- Improve content understanding.

## Featured Image

Every article should include:

- One featured image.
- SEO-friendly file name.
- Alt text containing the primary keyword naturally.

## Captions

Use captions when they add value to readers.

Avoid unnecessary captions.

## Final Goal

Use images to improve readability, accessibility, user experience, and SEO while maintaining fast page speed.
