# Keyword Rules

## Primary Keyword Placement

Ensure the primary keyword appears naturally in:

* SEO Title
* Meta Description
* URL Slug
* H1
* At least one H2
* H3 headings when relevant
* H4 headings when appropriate
* The first 100 words
* Conclusion
* At least one image alt text

Avoid forcing keywords unnaturally.

## Keyword Density

* Maintain approximately 1.5% to 3% keyword frequency.
* Prioritize readability over exact density.
* Avoid keyword stuffing.
* Write naturally for humans.

## Secondary Keywords

Include secondary keywords naturally throughout the content.

Place them in:

* H2 headings
* H3 headings
* Body content
* FAQ section

## Semantic SEO

Include:

* Related entities
* NLP keywords
* Synonyms
* Contextually relevant terms

Cover the topic comprehensively.

## Long-Tail Keywords

Incorporate long-tail keywords naturally where appropriate.

Avoid creating separate sections solely for keyword insertion.

## Search Intent

Identify and satisfy:

* Informational intent
* Commercial intent
* Transactional intent
* Navigational intent

Content should completely satisfy user intent.

## Featured Snippets

Increase featured snippet opportunities using:

* Definitions
* Bullet points
* Numbered lists
* Tables
* Comparison tables

## FAQ Optimization

Include commonly searched questions.

Use:

* People Also Ask questions
* Related queries
* Long-tail variations

## Anchor Text Optimization

Use descriptive anchor text.

Avoid:

* Click here
* Read more
* Learn more

Use keyword-rich but natural anchor text.

## Keyword Usage Rules

Do:

✓ Write naturally.

✓ Use variations of keywords.

✓ Use synonyms.

✓ Use entity-based optimization.

✓ Optimize for topical authority.

Avoid:

✗ Keyword stuffing.

✗ Repeating exact phrases excessively.

✗ Unnatural keyword placement.

✗ Sacrificing readability for keyword density.

## Final Goal

Create content that ranks naturally, satisfies search intent, demonstrates topical authority, and provides an excellent user experience.
