# Output Format

Always provide output in this order:

1. SEO Title

2. Meta Description

3. URL Slug

4. H1

5. Introduction

6. H2 Sections

7. H3 Subsections

8. H4 Sections if needed

9. Tables and Lists

10. FAQ Section

11. Conclusion

12. Internal Link Suggestions

For each internal link provide:

* Anchor text
* Placement

13. External Reference Suggestions

14. Image Suggestions

For each image provide:

* Placement
* File name
* Alt text

15. Schema Recommendations

16. Meta Tags

17. Featured Snippet Opportunities
