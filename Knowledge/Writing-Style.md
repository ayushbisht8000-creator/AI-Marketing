# Writing Style

## Reading Level

* Write at a 7th–8th grade reading level.
* Use simple and easy-to-understand language.
* Avoid unnecessary jargon.

## Tone

* Professional
* Helpful
* Trustworthy
* Conversational
* Human-written

## Paragraph Structure

* Keep paragraphs short (2–4 lines).
* Avoid large blocks of text.
* Improve readability and engagement.

## Sentence Structure

* Use active voice whenever possible.
* Mix short and medium-length sentences.
* Avoid repetitive sentence patterns.

## Human Writing Rules

Avoid:

* Robotic AI language
* Generic introductions
* Repetitive transitions
* Keyword stuffing
* Overly formal language

Write as if an experienced professional is explaining the topic to a reader.

## Formatting

Use:

* Bullet points
* Numbered lists
* Tables
* Comparison tables

When they improve readability.

## Engagement

* Address reader questions naturally.
* Explain concepts clearly.
* Use examples where appropriate.
* Focus on solving reader problems.

## SEO Content Rules

* Prioritize search intent.
* Write for users first.
* Include semantic relevance naturally.
* Maintain topical authority.

## Content Quality

Every piece of content should be:

* Helpful
* Accurate
* Easy to scan
* Actionable
* Well-structured
* Human-focused

## Final Goal

Create content that feels genuinely human-written, easy to read, engaging, and optimized for both users and search engines.
